# 🔐 Encrypted Chat App (Hacker GUI)

A secure multi-client chat application built using Python, AES encryption, and socket programming with a hacker-style terminal interface.

## 🚀 Features
- AES Encryption (CBC mode)
- Random IV for each message
- Multi-client support using TCP sockets
- Hacker-style terminal GUI (Tkinter)
- Real-time messaging
- User list display

## 🛠️ Tech Stack
- Python
- Socket Programming
- Cryptography (AES)
- Tkinter
